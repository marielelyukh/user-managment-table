# User Table App

Angular 19 application with a user table featuring infinite scroll, filtering, sorting, and search functionality.

## Features

- User table with columns: First Name, Last Name, Date of Birth, Phone Number, Active status
- Infinite scroll functionality
- Text search with highlighting
- Filters for active/inactive users and age groups
- Sorting for First Name, Last Name, and Date of Birth
- IndexedDB for data storage
- Ant Design UI components

## Installation

```bash
npm install
```

## Development

```bash
npm start
```

The application will be available at `http://localhost:4200`

## Build

```bash
npm run build
```
